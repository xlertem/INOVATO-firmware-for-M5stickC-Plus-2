# Assets for Figma export
Place your exported Figma assets (PNG/SVG) here before building.
Recommended structure:
- data/assets/screen_main@1x.png
- data/assets/icon_wifi.png
- data/assets/btn_top_idle.png
- data/assets/btn_top_pressed.png
- data/assets/battery_bar.png

If you use PlatformIO's "Upload File System Image" feature, these files will be included in SPIFFS/LittleFS.
