# M5StickC Plus2 - Figma-based Firmware (PlatformIO project)

Это готовый проект PlatformIO, подготовленный для сборки прошивки под M5StickC Plus2 (экран 135x240).
ВНИМАНИЕ: В этой среде я не могу собрать бинарник (.bin) сам — однако этот репозиторий содержит всё необходимое
для автоматической сборки через GitHub Actions или локально на вашем компьютере.

## Как получить .bin (вариант A) — GitHub Actions (рекомендуется)
1. Создайте новый репозиторий на GitHub (публичный или приватный).
2. Скопируйте все файлы из этого архива в репозиторий (включая скрытую папку .github).
3. Закоммитьте и запушьте в ветку `main`.
4. Зайдите в Actions на GitHub — workflow `Build M5StickC-Plus2` запустится автоматически.
5. После завершения выполните переход в Run → Artifacts → скачайте `firmware` (firmware.bin).

## Как собрать локально (вариант B) — PlatformIO
1. Установите VS Code и PlatformIO extension.
2. Откройте эту папку как проект PlatformIO.
3. Поместите экспортированные ассеты из Figma в `data/assets/`.
4. Нажмите "PlatformIO: Build" — после успешной сборки бинарник будет в `.pio/build/m5stickc_plus2/firmware.bin`.
5. Загрузите .bin на устройство через PlatformIO Upload или используйте esptool/web-flasher.

## Примечания
- Перед сборкой убедитесь, что `lib_deps` в platformio.ini корректно установлены.
- Замените заглушки в `src/main.cpp` при необходимости на реальные API считывания батареи из M5.Axp или M5.Power.
- Если хотите — могу подготовить и запушить файлы в ваш GitHub (нужен доступ), или помочь шаг за шагом.
